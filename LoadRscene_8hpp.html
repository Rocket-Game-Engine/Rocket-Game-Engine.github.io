Detailed Description

This is the file responsible for loading scenes into the engine. It works entirely dynamically, assuming that any components

  1. Are initialized before the call
  2. Are defined with public setters in the lambda map _setters
  3. Have a static member function name() that return the name of the component

This is all achieved through the macros defined in Component.h
