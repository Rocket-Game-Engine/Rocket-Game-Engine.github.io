Detailed Description

This file contains all the classes and functions revolving around textures. The end user mostly won't work with textures except through loading them via Rscene and accessing the TextureInfo of textures. Outside of that, the Sprite component usually takes care of the rendering automatically.

This is how the user is intended to load a texture into Rocket. It will be loaded from the filename (a path relative to the location the app was called from) and store the generated TextureInfo object into Textures::TextureMap. Then, it will return the TextureInfo object.

Parameters
